Section Two

Brand Momentum Phase

Why This Phase is Required

FiLLi Cafe's online ordering infrastructure (Cofe Cloud) does not pass conversion data back to Meta. Running Sales/Conversion campaigns is like driving with no speedometer — spending money with zero ability to measure return.

Stop optimizing for conversions we can't track. Build the audience, brand recall, and data infrastructure that will make future Sales campaigns 3-5x more effective.

How It Protects Media ROI

  • Eliminates wasted spend on broken conversion optimization — was losing AED 2,100 in 14 days with 0 tracked purchases
  • Redirects budget to objectives Meta CAN optimize for — reach, video views, engagement are all trackable
  • Builds retargeting audiences from 1,370 to 100,000+ before re-enabling Sales

How It Prepares for Future Conversion Scaling

  • Large warm audiences = lower CPA when Sales campaigns return
  • Video viewer pools = highest-intent retargeting segments
  • Brand recall lift = more organic/direct orders via Cofe Cloud dashboard
  • App install base = direct push notification channel that bypasses Meta entirely

Targeting Logic

  • Broad first — Let Meta's algorithm find the best audience within the geo radius. Data shows broad outperformed interest targeting.
  • Geo clusters — 7km radius around all 40 UAE stores. Covers major population centers without overlap.
  • No Advantage+ Audience — Always set advantage_audience: 0. We control targeting.
  • No Audience Network — Always exclude. Publisher platforms: Facebook + Instagram only.
  • Age bands — 18-55 for awareness, 21-45 for app installs (purchasing demographic).

Scaling Triggers

  • Campaign hits benchmark KPIs for 7 consecutive days → increase budget 20%
  • CPM drops below AED 2.00 → opportunity to scale aggressively (cheap reach)
  • ThruPlay cost drops below AED 0.01 → double video budget
  • Never increase budget more than 30% in a single day (avoids learning phase reset)

Placement Control

  • Phase 1 (Week 1-2): Manual — FB Feed, FB Reels, IG Feed, IG Reels only
  • Phase 2 (Week 3-4): Add IG Stories, FB Stories if Phase 1 delivers
  • Phase 3 (Month 2+): Test Advantage+ placements EXCLUDING Audience Network
  • NEVER enable Audience Network
